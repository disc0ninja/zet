# Immutable infrastructure

## Benefits  
- Testable infrastucture 
- Reproduce production 
- Unit of deployment 
- Confident changes  
  

## Cattle vs Cats: 

### Cat (server1)  
- Treated like a pet.
- If it runs away (server dies) it's difficult/impossible to replace 
  
### Cattle (server{1..n}}) 
 
- All the same
- Expendable and easily replaced
