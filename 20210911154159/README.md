# tfenv

[Terraform](https://github.com/disc0ninja/zet/search?q=terraform) version manager

## Useful Commands
``tfenv list``  
  - *installed* versions
  
``tfenv list-remote``  
  - versions available to be installed  

``tfenv install <version>``
  - Options Include:
    - latest
    - 0.7.0
    - min-required
    - latest:^0.8  
  
``tfenv use <version>``  
  - Options Include:
    - latest
    - 0.7.0
    - min-required
    - latest:^0.8  

``tfenv uninstall <version>``  
  - Options Include:
    - latest
    - 0.7.0
    - min-required
